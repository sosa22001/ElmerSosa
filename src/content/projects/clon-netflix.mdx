# 🎬 Clon de Netflix

## Descripción
Aplicación inspirada en **Netflix**, con:
- 🔍 **Exploración de contenido** por categorías.
- 👥 **Perfiles de usuario personalizados**.
- ⭐ **Favoritos y lista de reproducción**.
- 🔐 **Autenticación segura** con JWT.

## Tecnologías utilizadas
- 🟢 **Node.js**
- 🔴 **MongoDB**
- 🔵 **React**
- 🟡 **Tailwind CSS**
