# 🛒 Mercadito NUBIRA

## Descripción
Sistema web para **comerciantes** que permite:
- 📦 **Gestión de inventarios** y control de stock.
- 💰 **Registro de ventas** y facturación.
- 📊 **Reportes de negocio** con análisis de datos.

## Tecnologías utilizadas
- 🔵 **TypeScript**
- 🟢 **Node.js + Express**
- 🟣 **SQL Server**
- 🟡 **React + Firebase Storage**
