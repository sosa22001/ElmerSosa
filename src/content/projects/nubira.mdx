# 🛒 Mercadito NUBIRA

## Descripción
Sistema web diseñado para **comerciantes**, con funcionalidades como:  
- 📦 **Gestión de inventarios** y control de stock.  
- 💰 **Registro de ventas** y facturación automatizada.  
- 📊 **Reportes de negocio** con análisis de datos en tiempo real.  

## Tecnologías utilizadas
- 🔵 **TypeScript**  
- 🟢 **Node.js + Express**  
- 🟣 **SQL Server**  
- 🟡 **React + Firebase Storage**
