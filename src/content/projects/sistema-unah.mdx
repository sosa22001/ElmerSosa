# 🏫 Sistema de la UNAH

## Descripción
Plataforma académica diseñada para la **gestión de estudiantes y docentes**, con funcionalidades como:  
- 📖 **Proceso de matrícula automatizado** para agilizar inscripciones.  
- 👩‍🏫 **Administración de docentes y asignaturas** con gestión eficiente.  
- 🔐 **Sistema de roles y permisos** para garantizar seguridad.  

## Tecnologías utilizadas
- 🟢 **Node.js**  
- 🔵 **TypeScript**  
- 🟣 **SQL Server**  
- 🟡 **React + Tailwind CSS**
