# 🏫 Sistema de la UNAH

## Descripción
Plataforma académica para la **gestión de estudiantes**, con:
- 📖 **Proceso de matrícula automatizado**.
- 👩‍🏫 **Administración de docentes** y asignaturas.
- 🔐 **Sistema de roles y permisos** para mayor seguridad.

## Tecnologías utilizadas
- 🟢 **Node.js**
- 🔵 **TypeScript**
- 🟣 **SQL Server**
- 🟡 **React + Tailwind CSS**
