# 🌍 WebApp Congreso UNAH

## Descripción
Plataforma web para la **gestión del Congreso de Innovación 2025**, con:
- 📌 **Registro de usuarios** y autenticación segura.
- 🎟️ **Validación de asistencia mediante QR**.
- 📅 **Administración de eventos en tiempo real**.

## Tecnologías utilizadas
- 🟢 **Node.js**
- 🔵 **TypeScript**
- 🟣 **React**
- 🟡 **Firebase**
