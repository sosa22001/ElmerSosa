# 🖥️ WebApp Congreso UNAH

## Descripción
Desarrollo de una plataforma web para la gestión del Congreso de Innovación 2025, un evento con ponentes nacionales e internacionales. La WebApp facilita la administración de conferencias, talleres, usuarios y validación de asistencia mediante QR para un estimado de 500 usuarios. 

## Tecnologías utilizadas
- 🟢 **Node.js**  
- 🔵 **TypeScript**  
- 🟣 **Next.JS**  
- 🟡 **Supabase**
